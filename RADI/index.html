RADI: LLMs as World Models for Robotics Action Decomposition and Imagination

Weixing Guo

A novel robotic system using LLMs for action decomposition and imagination

Abstract

This project introduces RADI, a novel robotic system that leverages Large Language Models (LLMs) as world models for action decomposition and imagination. The system aims to enhance robotic task planning and execution through advanced AI techniques. Key features include:

- LLM-based action decomposition
- World model simulation and imagination
- Task planning and execution
- Integration with robotic control systems
