Data Transmission and Path Planning for UAV in Precise Agriculture

Weixing Guo*, Qiyuan Liu*,
In progress(Plan to publish in ICRA 2026)

Real Implementation Videos

Abstract

本研究专注于为精确农业应用开发高效的UAV路径规划算法。通过结合先进的路径规划技术和农业特定需求，我们提出了一种能够优化数据收集效率、减少能源消耗并提高监测精度的解决方案。

This research focuses on developing efficient UAV path planning algorithms for precision agriculture applications. By combining advanced path planning techniques with agricultural-specific requirements, we propose a solution that optimizes data collection efficiency, reduces energy consumption, and improves monitoring accuracy.
